# ADR Template

The canonical ADR template is maintained in [homelab-docs](https://github.com/jbcodeforce/homelab-docs/blob/main/adrs/0000-template.md).

When creating a new ADR for this repository, copy the template from homelab-docs:

```bash
curl -o adrs/NNNN-short-title.md https://raw.githubusercontent.com/jbcodeforce/homelab-docs/main/adrs/0000-template.md
```

See the [ADR README](README.md) for full instructions.
